Major projects

During the past 4 years I've designed and developed an open-source data framework broadly titled ZincBase, split into:

  • ZincDB: A JavaScript database and real-time synchronization library which includes features like offline storage, background execution within workers, hierarchal data organization, end-to-end encryption (TypeScript, ~16,000 LOC, MIT License).
  • ZincServer: Independent server component which bundles a custom on-disk transactional storage engine with a high-performance HTTP server (Go, ~6,000 LOC, Apache 2.0 License).
  • ZincEditor is a corresponding web-based, real-time database editor in an early stage of development and is based on a React/Redux like architecture (TypeScript, ~500 LOC without shared library code, MIT License).

Their combined feature-set intersects with software and cloud services like Firebase (real-time single and multi-user data sync), PouchDB (in-browser and offline storage abstractions) and Apache Kafka (real-time server-side message storage and dispatch). The project has been published but should be seen as mostly experimental in nature. The client-side library is planned to be modularized into multiple smaller independent libraries, which would be gradually adapted and prepared for production use.

LZ-UTF8 (TypeScript ~4,000 LOC) is a high-performance string compression library and stream format. The library is in active use.
